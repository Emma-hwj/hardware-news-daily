# 计算机硬件资讯日报

这是一个自动化收集和整理计算机硬件领域最新资讯的项目。每天从多个渠道获取最新资讯，包括：

## 资讯来源
- 国外网站：
  - TechCrunch
  - The Verge
- 国内网站：
  - 雷锋网
  - 机器之心

## 关注领域
- 服务器
- 核心部件（CPU、GPU、内存、硬盘）
- AI云计算
- AI应用

## 文件结构
- daily_reports/: 存放每日资讯报告
- scripts/: 自动化脚本
- templates/: 报告模板

## 更新频率
每天更新一次，文件命名格式为：`计算机硬件资讯日报_yyyy-mm-dd.md`
